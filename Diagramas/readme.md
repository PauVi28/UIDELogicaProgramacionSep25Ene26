## Contenido

En esta carpeta se almacenan los diagramas de flujo correspondientes a cada una de las funcionalidades del sistema, representando de manera gráfica la lógica y el orden de los pasos necesarios para resolver cada problema planteado.

## Simbología utilizada

Los diagramas de flujo siguen la simbología estándar:

- **Óvalo:** Inicio / Fin
- **Rectángulo:** Proceso
- **Rombo:** Decisión
- **Paralelogramo:** Entrada / Salida de datos
- **Flechas:** Indican el flujo o secuencia del proceso
